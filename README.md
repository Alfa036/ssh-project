# SSH və Server Tapşırığı
1) Linux server quraşdırıldı.
2) SSH bağlantısı aktiv edildi.
3) ssh-keygen ilə RSA açarları yaradıldı.
4) Public key `authorized_keys` faylına əlavə edildi.
5) SSH konfiqurasiyası (`sshd_config`) tənzimləndi.
6) `.ssh/config` faylı yaradıldı və alias təyin edildi.
7) GitHub repository yaradıldı.
8) Bütün proses README de qeyd edildi.
